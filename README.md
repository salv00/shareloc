Backend du projet ShareLoc, une application de gestion de colocations permettant aux utilisateurs de partager des services, voter et suivre les tâches au sein d’une colocation.

Ce projet a été réalisé dans le cadre d’une Licence Professionnelle en Développement d’Applications 2022-2023, à titre pédagogique.
L’objectif est de mettre en pratique les compétences acquises en Java, Spring Boot, bases de données relationnelles et API REST.

Technologies utilisées : 

Java 17
Spring Boot 2.7+
Spring Security + JWT
Spring Data JPA / Hibernate
PostgreSQL
Maven
Lombok
Spring Web
Spring Validation

Fonctionnalités principales

Authentification & autorisation avec JWT
Gestion des utilisateurs et colocations
Ajout, validation et vote des services
Upload d’images (multipart/form-data)
Gestion des rôles (Admin / Colocataire)
API REST sécurisée et structurée
